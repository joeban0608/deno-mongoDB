---
## 專案啟動步驟：

- 啟動 deno-api
    
    ```jsx
    cd deno-api
    deno run --allow-net app.ts
    ```
    
- 啟動 frontend-app
    
    ```jsx
    cd frontend-app
    npm install
    npm run start
    ```
    
- 啟動 mongo
    
    ```jsx
    docker compose up -d
    ```
